## 2. 기술 아키텍처 (현재 → 목표)

### 현재 (작동 중)
```
[프론트] HTML/CSS/JS + Lucide 아이콘   (정적)
[백엔드] Supabase = PostgreSQL + Auth + RLS(보안정책)
[기능]   신청·주문번호·비회원조회·관리자칸반·입고/배송비·공지·센터주소·요율표
```

### 목표 (프리미엄·안전)
```
[배포]   Cloudflare Pages (무료 HTTPS·CDN·DDoS방어)
[보안]   RLS + Cloudflare Turnstile(봇차단) + 비밀키는 서버에만
[서버]   결제검증·알림·환율·파싱 = 서버함수(Python:Vercel / TS:Supabase)
[결제]   포트원(PG) — 서버검증 필수
[알림]   카톡 알림톡(솔라피) — 상태변경 자동
[저장]   Supabase Storage — 검수사진
```

- [x] **P1 — 회원 + 등급 + 권한 + 로그인** (구축 완료, Supabase SQL 실행 대기)
  - 아이디 기반 회원가입/로그인 (아이디 → 내부 `아이디@oss.local` 매핑)
  - 등급(역할): 마스터관리자 / 매니저관리자 / 일반회원 + 마스터가 매니저 세부권한 부여
  - 로그인 시 "아이디 틀림 / 비번 틀림" 팝업, 등급별 이동(운영진→admin, 회원→mypage)
  - 마이페이지(내 정보·주문내역·배대지 사서함 코드), 관리자 회원관리 탭
  - 회원등급 사다리(guest/silver/gold/diamond/red/biz) 필드 마련 — 혜택은 추후
- [ ] **P2** — 예치금/적립금 지갑 + 결제내역 + 사서함 코드 자동발급 활용 + 회원 메시지/탈퇴내역
- [ ] **P3** — 주문관리 상태머신 확장(도착스캔→입고→포장/무게측정→결제대기→배송→완료, 랙넘버, 반송/교환, 휴지통)
- [ ] **P4** — 신청유형 확장(메루카리 전용 구매대행, 일괄등록, 재고관리)
- [ ] **P5** — 컨텐츠/프로모션/배너/쿠폰, 정산·통계, 배송스케쥴, 번역기, SMS·알림
- [ ] **P6** — 소셜로그인(Kakao), 관세청 고시환율 자동, 송장 대역대 등 세부

### P1 적용에 필요한 사장님 액션 (Supabase)
1. SQL Editor에서 `supabase-members.sql` 실행 (profiles 테이블·트리거·RLS 생성)
2. Authentication → Providers → Email → **"Confirm email" OFF** (아이디 가입은 가짜 이메일이라 인증메일 못 받음)
3. 본인 계정을 마스터로: `update profiles set role='master' where email='oss202604@gmail.com';`

## 🧾 운영 준비물 (코드 아닌 것 — 실제 사이트 운영에 필수)

- ⬜ **결제(PG) 연동**: 카드/간편결제 받으려면 PG사 계약(토스페이먼츠/이니시스 등) + 사업자 필요. 지금은 무통장/계좌이체 안내 수준.
- ⬜ **예치금 충전 처리**: 무통장 입금확인(수동) 또는 PG 자동. (사토리 = 예치금 충전/무통장 내역 메뉴 있음)
- ⬜ **SMS/카카오 알림톡**: 발송업체(솔라피/알리고/카카오 알림톡) 계약 + 발신번호 사전등록.
- ⬜ **소셜로그인(Kakao)**: 카카오 개발자 앱 등록 + REST 키 + 도메인 등록.
- ⬜ **메루카리 API**: clientid/secret 발급(파트너 승인). 자동구매용.
- ⬜ **일본 현지 배대지 주소/창고**: 실제 오사카 등 주소·검수 인력 확보(회원 사서함 발급의 전제).
- ⬜ **사업자/법무**: 사업자등록, 통신판매업 신고, 개인정보처리방침·이용약관(보유), 해외직구·통관 고지.
- ⬜ **관세청 고시환율 자동수집**: 주간 고시환율 크롤링/API → 예상비용·정산 반영.
- ⬜ **호스팅/도메인/요금**: 도메인 연결, HTTPS, Supabase 유료플랜(트래픽·용량·백업), service_role 키 절대 비공개.
- ⬜ **CS 운영**: 카톡 채널 개설, 응대시간, 공지/FAQ 최신화.
