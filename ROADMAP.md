## 2. 기술 아키텍처 (현재 → 목표)

### 현재 (작동 중)
```
[프론트] HTML/CSS/JS + Lucide 아이콘   (정적)
[백엔드] Supabase = PostgreSQL + Auth + RLS(보안정책)
[기능]   신청·주문번호·비회원조회·관리자칸반·입고/배송비·공지·센터주소·요율표
```

### 목표 (프리미엄·안전)
```
[배포]   Cloudflare Pages (무료 HTTPS·CDN·DDoS방어)
[보안]   RLS + Cloudflare Turnstile(봇차단) + 비밀키는 서버에만
[서버]   결제검증·알림·환율·파싱 = 서버함수(Python:Vercel / TS:Supabase)
[결제]   포트원(PG) — 서버검증 필수
[알림]   카톡 알림톡(솔라피) — 상태변경 자동
[저장]   Supabase Storage — 검수사진
```
