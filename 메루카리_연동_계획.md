# 메루카리 실시간 연동 — 실행계획 (멀티에이전트 설계 2026-06-23)

> 아키텍트3 → 크리틱 → 플래너 워크플로우(7에이전트) 종합. 핵심: 크리틱이 **"싼 줄 알았던 상시폴링 = 월 ₩300만 폭탄"**을 적발 → 온디맨드 방식으로 재설계.

## 핵심 결정
- **데이터취득 = Apify 기성 메루카리 액터**(예: fatihtahta/mercari-japan-scraper) 1개. 우리 Edge Function은 메루카리를 직접 안 때리고 **Apify만 호출** → IP차단·DPoP·Cloudflare·유지보수 회피(깨지면 액터 제작자가 대응, 안되면 actorId 한 줄 교체). **자체 스크래핑(DPoP 직접구현)은 폐기** — 1인 컴맹+정적사이트엔 자살(TLS지문·150초 한계·빈화면 자가복구 0%).
- **방식 = 온디맨드 우선**: 손님이 검색/클릭한 **순간만** 가져와 캐시(검색 TTL 15분·상세 10분). 상시 카탈로그 폴링 ❌(과금 폭탄). ※변경감지 해시는 과금 못 줄임(스크랩 후 DB단에서 발생) — 번역/이미지 중복방지용으로만.
- **번역 = Google Cloud Translation**(JA→KO). DeepL Free는 약관상 "유료 손님서비스 사용 금지"라 부적합. translation_cache 영구저장 + 셀러용어(新品/未使用/美品 등) 수동시드로 비용 0 수렴.
- **이미지 = Supabase Storage 미러**(썸네일 static.mercdn.net 핫링크 403 실측). 상세진입분 대표 1~3장만 프록시 다운로드 저장.
- **비용 자물쇠**: Apify 계정 spend-limit($30/월) + 코드 하드캡(큐레이션 5키워드·6시간·15건) + mercari_sync_log 일일가드 → **월 ₩5~13만 강제고정**(폭주 자동차단).
- **"완전 실시간" 불가**(메루카리 공식 웹훅/피드 없음) → **"클릭=그 순간 최신가·재고 확인"**으로 재정의(3층 신선도: 온디맨드>검색TTL캐시>인기키워드 사전캐시).

## 비용 (정정 — 중요)
- 오픈초기 월 **약 ₩5~13만** (Apify spend-limit이 상한 강제). Google번역 통상 ₩1만↓, Supabase Free 시작가능.
- ⚠️ **상시폴링 넣으면 월 ₩300만**까지 튐 → 절대 채택 안 함.

## 단계
- **Phase 0 (mvpFirstStep · ₩0 · 코드/DB 0줄)**: 사장님 Apify 무료가입(이메일만, **카드 불필요**)→API토큰 발급→Claude 전달. Claude가 무료체험 2000건으로 키워드 2~3개+상세URL 1~2개 테스트 → ①필드(제목·가격·재고·사진·설명·상태) 다 오나 ②사진 403/200 ③실제 월비용 측정 → 1장 보고. **"쓸만"하면 진행, 아니면 매몰비용 0.**
- **Phase 1**: Supabase 4테이블(mercari_items·mercari_searches·translation_cache·mercari_sync_log)+Storage(mercari-img)+settings. Edge Function /search·/detail(캐시조회→미스시 Apify→upsert→Google번역). 일일 비용가드 코드내장. 🙋사장님: Google Cloud Translation 키 발급(Claude 가이드) + Apify spend-limit $30 설정.
- **Phase 2**: 기존 헤더/푸터 복사로 mercari.html(검색+결과그리드)·product.html?id=(상세, 재팬큐 /deal/detail 모사) 신설 + 하단 '구매대행 신청'→order-quick.html prefill로 기존 applications 흐름 연결.
- **Phase 3 (선택)**: pg_cron이 인기키워드 3~5개만 하드캡 사전캐시 → 홈 '메루카리 신착' 띠. 액터 헬스체크 403률↑시 mercari_enabled 자동OFF('점검중').

## 🙋 사장님 결정/액션
1. **월 5~10만원대 고정비 기능 + 완전실시간 불가(클릭시 근사)** 이해하고 진행 결정
2. 약관 회색지대 인지 — '구매대행 중개' 포지셔닝, 차단·중지요청 시 즉시 OFF 스위치 내장
3. **Apify 무료가입 + API토큰**(Phase 0, 카드 불필요)
4. **Apify Spend limit $30/월** 직접 설정(폭주 자동차단 — 사장님만 가능)
5. **Google Cloud Translation 키**(Phase 1, Claude 스크린샷 가이드)
6. 인기키워드 3~5개(Phase 3, 선택) · 출처/면책 문구 승인
