# 경쟁사·고객 니즈 분석 → OSS에 추가/폐기할 것

> 조사 대상: 사토리, 몰테일, 델리버드, 텐소, 제이포스트, 사줘, 헤이재팬, 재팬타임몰 등
> 핵심 결론: **고객의 진짜 불안 = "내 물건 안전한가 / 지금 어디 있나 / 통관 되나 / 문제 생기면 보상되나"**
> → 이 불안을 없애는 기능 = 곧 신뢰 = 재구매. 몰테일이 분실·파손 책임회피로 평판 추락한 게 반면교사.

---

## ⭐ 추가할 것 (고객가치 큰 순)

### 신뢰·안심 (최우선 — 경쟁 차별화)
1. **검수 사진** — 입고 시 사진 업로드 → 고객이 조회에서 확인. "사기·파손 불안" 1차 해소 (사기방지 = 구매대행 핵심 가치)
2. **실시간 처리상태 + 자동 알림(카톡/이메일)** — "내 물건 어디?" 불안 해소. 상태 바뀔 때마다 자동 통보 = 문의 70%↓
3. **분실·파손 보상정책 명시 + (선택)보험옵션** — 몰테일 반면교사. "문제 생기면 보상" 명시가 신뢰 차별점
4. **통관 안내·개인통관부호 검증** — 통관 불안 큼. 안내 명확 + 부호 형식 체크

### 비용 투명성 (구매대행 핵심 만족도)
5. **예상비용 계산기** — 상품가+수수료+배송비 미리 확인 (숨은 비용=불신)
6. **부피무게 자동계산** — 실무게 vs 부피무게 중 큰 값 기준(업계 표준). 요율계산에 반영 필요 ⚠️ 현재 실무게만 → 보강
7. **합배송** — 트래킹 N개까지 무료(업계 5박스 관행) → 비용 절감 어필

### 편의 (전환율↑)
8. **상품 URL 붙여넣기 → 자동 정보/번역** — 신청 편의 (조이포스트 실시간 번역 강점)
9. **카카오 우편번호 주소검색** — 입력 편의 (무료)
10. **회원등급 할인 + 적립** — 재구매 유도 (사토리도 운영)
11. **모바일 최적화 / PWA(앱처럼)** — 몰테일 "앱 할인"으로 리텐션. 무료로 앱느낌 가능

### 신뢰 표시 (정식 업체로 보이게)
12. **실시간 처리현황·후기·누적건수** 노출
13. **사업자정보·통신판매업번호·약관·환불정책** (법적+신뢰)

## 🗑️ 폐기 / 보류할 것 (우리 규모엔 과함)

- ❌ **다국가 창고** (몰테일식) — 우리는 일본 전문. 선택과 집중
- ⏸️ **복잡한 예치금 다단계 / 과한 등급·쿠폰 체계** — 초기엔 단순하게, 나중에
- ⏸️ **일괄등록(엑셀)·재고관리** — 대형 셀러(B2B)용. 일반고객엔 불필요 → 후순위
- ⏸️ **경매 대리입찰** (텐소식) — 수요 확인 후
- ❌ **과도한 팝업·배너** — 싸구려 느낌. 깔끔 유지

---

## 💡 OSS 차별화 포인트 (이걸로 승부)
1. **검수 사진 + 실시간 알림** = "믿고 맡기는" 안심 경험
2. **투명한 예상비용** (부피무게까지 정확) = 숨은비용 0
3. **깔끔한 디자인 + 자연스러운 한국어** = 정식 업체 신뢰감
4. **빠른 카톡 CS** = 응답속도 = 만족도

> 한 줄: "사기 걱정 없이, 내 물건이 지금 어디 있는지 훤히 보이고, 비용이 투명한 일본 구매대행" = OSS
