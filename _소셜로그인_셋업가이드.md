## 🥇 1단계: 카카오 로그인 (쉬움 — Supabase 기본 지원)

### 👤 사장님이 할 것
1. **developers.kakao.com** 로그인
2. 상단 **「내 애플리케이션」** → **「+ 앱 생성」**
   - 앱 이름: **오쓰**
   - 회사명: **안승스토어**
   - 약관 동의 ✓ → **저장**
3. 만든 **「오쓰」 앱 클릭** → 들어가기
4. 왼쪽 **「앱 키」** → **REST API 키** 복사해두기 ⭐(핵심)
5. 왼쪽 **「카카오 로그인」** → **활성화 설정 ON**
6. 같은 화면 **Redirect URI** 에 등록:
   ```
   https://wmgzggeklwzhrlmpuifh.supabase.co/auth/v1/callback
   ```
7. **「카카오 로그인 ＞ 보안」** → **Client Secret 코드 생성** + **활성화 상태 ON** → 코드 복사
8. **「카카오 로그인 ＞ 동의항목」** → **닉네임**(필수)·**카카오계정(이메일)** 설정
9. (배포 후) **「플랫폼」 ＞ Web** → 사이트 도메인 등록(Netlify 주소). 지금 테스트는 `http://localhost:5500`

### 📤 클로드한테 줄 것: **REST API 키** + **Client Secret** (2개)

### 🤖 클로드가 할 것
- Supabase ＞ Authentication ＞ Providers ＞ **Kakao** 켜고 키 입력
- 로그인 페이지 **카카오 버튼** → 실제 로그인 연결(`signInWithOAuth`)
